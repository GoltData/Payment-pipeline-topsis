# Реализация высокоэффективного платежного конвейера в рамках хакатона XMAS-HACK 2024
# Постановка задачи 
<p align="center">
<img src="https://github.com/user-attachments/assets/dd62c1f5-de30-448e-96e8-fc097ecff112" width="800" />
  
При осуществлении транзакции необходимо выбрать провайдер(банк более простыми словами), который проведёт данный платеж. Выбор провайдера зависит от большого кол-ва факторов: конверсия, время обработки, комиссия, ограничения на размер транзакции и тп. Наша команда должна была разработать автоматическую систему, определяющую наиболее подходящую последовательность провайдеров для каждой транзакции, оптимизируя все вышеперечисленные признаки
